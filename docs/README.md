## 快速定位

- Docker 或 VPN 无法连接：查看[部署手册的常见问题](../WINDOWS-CONTAINER-VPN.md#常见问题)。
- 修改检查周期：查看[配置参考](CONFIGURATION.md#成绩检查与心跳)。
- 微信没有收到消息：查看[运维文档的推送检查](OPERATIONS.md#微信推送检查)。
- 转让或停用电脑：执行[隐私清除](OPERATIONS.md#清除全部隐私数据)。
- 理解为什么不会影响宿主机网络：查看[请求路径](ARCHITECTURE.md#请求路径)。
